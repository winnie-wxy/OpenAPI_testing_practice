Test Layers

  • Smoke (6) — Is the API alive? Auth works? Basic CRUD?
  • Regression (25) — Full lifecycle, negative cases, data variations, filtering
  • Contract (3) — Pydantic schema validation (catches field drift)
  • Non-Functional (14) — Response time P95, concurrency, security, resilience

Engineering Patterns

  • API client class — tests never call requests directly
  • Data factory — randomised, parallel-safe, no shared state
  • Yield fixtures — guaranteed cleanup on pass or fail
  • Environment config — .env + --env flag for multi-env
